# 🗳️ Election Online Voting System

A secure and scalable **online election platform** that enables users to vote digitally while ensuring transparency, integrity, and ease of management.

---

## 🚀 Overview

This project is a full-stack web application designed to conduct elections digitally. It allows administrators to manage elections and users to cast votes securely from anywhere.

---

## ✨ Features

### 👨‍💼 Admin Panel

* Create and manage elections
* Add / remove candidates
* Start and end elections
* Manage users (register, delete)
* View results and analytics

### 👤 User Features

* Secure login & authentication
* View available elections
* Cast vote (one vote per user)
* View results after election ends

### 🔐 Security Features

* Authentication & authorization
* One-person-one-vote validation
* Secure backend APIs

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/sumitraj7766/Election_online.git
cd Election_online
```

### 2️⃣ Install dependencies

```bash
npm install
cd frontend
npm install
```

### 3️⃣ Setup Environment Variables

Create a `.env` file in backend:

```
PORT=5000
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret_key
```

### 4️⃣ Run the project

```bash
# Run backend
npm start

# Run frontend
npm run client
```

## 🧠 How It Works

1. Admin creates an election
2. Admin adds candidates
3. Users register/login
4. Election starts
5. Users cast votes
6. Election ends
7. Results are displayed
